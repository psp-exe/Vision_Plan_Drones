Vision_Plan_Drones

Vision_Plan_Drones is a software package designed to enable autonomous navigation for unmanned aerial vehicles (UAVs). It bridges the gap between visual perception and flight control.
What It Does

This project integrates computer vision with path planning algorithms to accomplish three main tasks:

    Perception: Analyzes real-time camera feeds to detect obstacles, features, or targets in the environment.

    Planning: Generates optimal, collision-free paths through complex environments using algorithms like A* or RRT.

    Execution: Translates the planned paths into actionable flight commands for the drone controller.

Purpose

The goal of this repository is to provide a lightweight, modular framework for researchers and developers to test and deploy vision-based navigation strategies, suitable for both simulation environments and real-world drone hardware.
